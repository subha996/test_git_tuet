## create a new repository on the command line
```echo "# Wine-Quality-is-BAD" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/subha996/Wine-Quality-is-BAD.git
git push -u origin main
```

```git remote add origin https://github.com/subha996/Wine-Quality-is-BAD.git
git branch -M main
git push -u origin main
```
